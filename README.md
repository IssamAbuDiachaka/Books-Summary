## How I Take Notes

Each book summary is a separate Markdown file with this structure:

- **Overview**: Brief plot/non-fiction summary (spoiler-free).
- **Key Takeaways**: Bullet points of main ideas.
- **Favorite Quotes**: Highlighted quotes.
- **Personal Reflections**: How it applies to my life.
- **Rating**: Out of 10.

Feel free to suggest improvements!
